# FoxRedbook

Cross-platform, AOT-compatible .NET library for bit-perfect audio CD ripping.

Named for the Red Book specification (IEC 60908) that defines CD-DA. Scope is deliberately tight: PCM samples in, verified PCM samples out. No encoding, no metadata lookup, no file management.
